Enum Statement

Enums define a type with a fixed set of named values.

Basic Enum

enum Color { Red, Green, Blue }

Enum with Values

enum Status { Ok(int), Error(string), Pending }

Pattern Matching

match status { Status::Ok(code) => println("Success: {code}"), Status::Error(msg) => println("Error: {msg}"), Status::Pending => println("Waiting..."), }

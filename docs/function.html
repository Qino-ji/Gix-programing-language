Function Statement

Functions in Vix are declared using the fn keyword. They can take parameters and return values.

Basic Syntax

fn add(a: int, b: int) -> int { return a + b; }

Void Functions

Functions that don't return a value omit the return type:

fn printHello() { println("Hello, World!"); }

Function Overloading

Vix supports function overloading based on parameter types:

fn process(x: int) { /* ... */ } fn process(x: string) { /* ... */ }

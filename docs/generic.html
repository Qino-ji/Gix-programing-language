Generic Statement

Generics allow writing flexible, reusable code that works with multiple types.

Generic Functions

fn identity<T>(value: T) -> T { return value; }

Generic Structs

struct Box<T> { value: T } fn main() { let intBox = Box<int> { value: 42 }; let strBox = Box<string> { value: "hello" }; }

Constraints

fn compare<T: Comparable>(a: T, b: T) -> bool { return a.compare(b); }

Struct Statement

Structs are composite data types that group related values together.

Struct Declaration

struct Point { x: float, y: float }

Creating Instances

let origin = Point { x: 0.0, y: 0.0 }; let p = Point { x: 10.0, y: 20.0 };

Tuple Structs

struct RGB(int, int, int); let red = RGB(255, 0, 0);

Class Statement

Classes in Vix provide object-oriented programming capabilities with encapsulation and methods.

Declaring a Class

class Person { name: string; age: int; fn new(name: string, age: int) -> Self { return Self { name, age }; } fn greet(self) { println("Hello, I'm {self.name}"); } }

Inheritance

Classes can inherit from other classes:

class Employee : Person { id: int; fn new(name: string, age: int, id: int) -> Self { return Self { name, age, id }; } }
